# Laozi Website

This is the official website for the Laozi project, featuring a Daoist-inspired meme coin and AI-based divination tool.

## Pages

- `index.html`: Homepage
- `airdrop.html`: Airdrop page with task details
- `divination.html`: Laozi Oracle page for divination
- `contact.html`: Contact page with social media links

## Setup

To run the website locally, clone the repository and open `index.html` in your browser.
